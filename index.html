Datos personales
Pasos para obtener tu Chat ID:
1. Abre Telegram en tu celular o computador
2. En el buscador escribe: @STEAM3D_UIS_BCA_bot
3. Abre el chat y escribe /start
4. El bot te responde con tu Chat ID — copia ese número aquí
Si no tienes Telegram puedes dejar este campo vacío, pero no recibirás notificaciones.
Modalidad de impresión
Debes estar presente durante toda la impresion. Si no puedes supervisar, la solicitud sera cancelada y se continuara con el siguiente en la cola.
